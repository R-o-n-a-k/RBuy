## 🚀 About the Project

**RBuy** is a fully responsive e-commerce application built using **React** and powered by the **Fake Store API**. The project focuses on building a clean, scalable architecture while following modern React best practices such as reusable components, dynamic routing, and modular folder organization.

The application allows users to browse products by category, explore individual product details, search products, and experience a responsive shopping interface suitable for desktop and mobile devices.

This project was built to strengthen frontend architecture, routing, API integration, and component reusability.

---

## 🛠️ Technologies Used

- **Frontend:** React.js, JavaScript
- **Routing:** React Router DOM
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **API:** Fake Store API
- **Hosting & Deployment:** Netlify / Vercel

## 📌 Application Routes

| Route | Description |
|--------|-------------|
| `/` | Home Page |
| `/shop` | All Products |
| `/shop/:category` | Category Products |
| `/product/:id` | Product Details |
| `/cart` | Shopping Cart |
| `/auth` | Login / Register |

## 🧑‍💻 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/RBuy.git

cd RBuy
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Open your browser

```
http://localhost:5173
```

## 🌐 API Used

This project uses the **Fake Store API** for fetching products and categories.

```
https://fakestoreapi.com/
```

Example endpoints:

```text
GET /products

GET /products/categories

GET /products/category/:category

GET /products/:id
```
